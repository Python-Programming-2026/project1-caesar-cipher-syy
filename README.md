项目简介
本工具是一款基于Python的凯撒密码加密解密程序，支持智能暴力破解功能，通过文本可读性评分算法自动识别最可能的解密结果。

功能特性
1.加密功能：将明文按指定步长偏移加密 
2.解密功能：将密文按指定步长还原为明文 
3.智能破解：自动分析并输出最可能的解密结果 
4.全量破解：显示所有25种可能的解密结果

评分维度一：字母频率评分
原理说明：英文中各字母出现频率不同，正常文本中高频字母如e、t、a出现次数较多。

频率排序：etaoinshrdlcumwfgypbvkjxqz（按频率从高到低排列）

计分规则：排名越靠前的字母权重越高，e字母权重为26分，t字母权重为25分，依次递减，z字母权重为1分。

计算公式：频率得分等于每个字母的权重分乘以该字母出现次数的总和。

示例说明：文本the中，e出现1次得26分，t出现1次得25分，h出现1次得7分，总计58分。

评分维度二：常见单词匹配评分
原理说明：正确解密的文本通常包含常见英文单词。

单词列表：the、be、to、of、and、a、in、that、have、i、it、for、not、on、with、he、as、you、do、at

计分规则：每匹配一个常见单词加10分，自动去除标点符号进行匹配。

示例说明：文本hello world无匹配单词得0分，文本the world匹配the加10分。

评分维度三：字母比例评分
原理说明：正常英文文本中字母占比通常较高，乱码则包含较多非字母字符。

计算公式：字母比例得分等于字母数量除以文本总长度再乘以20。

示例说明：文本Hello字母占比100%得20分，文本H3ll0字母占比50%得10分，文本123无字母得0分。

使用示例
命令行交互
运行程序后显示欢迎界面和功能菜单，用户输入选项1即可进入加密功能，输入明文Hello World，输入步长为3，得到密文为。




注意事项
语言支持：仅支持英文字母，中文、数字、符号不加密。 文本长度：短文本少于10字符时评分可能不准确。 步长范围：有效步长为1至25，26等于无偏移。 大小写：加密后保留原文大小写格式。

算法优化建议
如需提高破解准确率，可考虑以下优化方向：

扩展单词词典：增加common_words中的单词数量。 双字母频率：加入th、he、in等常见字母组合分析。 词性标注：检查解密结果是否包含完整的主谓宾结构。 机器学习：使用语言模型进行更精准的可读性评估。
